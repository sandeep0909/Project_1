Memory Game

Memory game is a beginner level gave for kids to test their Memory This game is built by using these technologies as under: * HTML * CSS * javascript * jQuery * Responsive design

Rules of engagement

  • This is a two player game. Player A and Player B. Player A starts first.
  • Each player will gets two turns. If they are correct they can go on.
  • The game gets over after 12 turns.
  • There is a score attached to the players.
  • The player can play till they make a wrong move.
  • A move is wrong when the player cannot match the two cards.
  • If a move is wrong then the wrong cards in that move is erased.
  • The second player starts next.
  • Each match results in 1 point.
  • The player who scores the most is the winner

Minimum Viable product

  • Ability to show the memory cards with fadeIn/fadeOut
  • Ability to shuffle the cards randomly at the start of the game
  • Ability to have players play their turns
  • Ability to track the scores
